## Requirements

- Node v6.6
- Redis v3.2.x
- Postgresql v9.5.x
- ffmpeg
- yarn (`npm install yarn -g`)

##Installation
IMPORTANT: Save a copy of the example keys file as 'keys.js'

From within the root directory:
```
./database.setup.bash
yarn
```

## Usage

start postgres server and redis server (changes depending on OS)

```
webpack & npm start
```
