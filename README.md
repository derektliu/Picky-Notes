## Requirements

- Node v6.6
- Redis v3.2.x
- Postgresql v9.5.x


## Installation

From within the root directory:

```
npm install
npm install -g webpack
```
After psql has been installed:

```
psql -d postgres;
CREATE USER ubuntu WITH PASSWORD 'password';
CREATE DATABASE pickynotes;
```

## Usage

```
webpack
npm start
```
